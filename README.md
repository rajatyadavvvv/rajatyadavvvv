- 🔭 Currently building: **Chart Selector API** — a FastAPI service that auto-selects and renders chart types from CSV data, deployed on **AWS ECS Fargate**

- 🛠️ Recently shipped: **Medallion architecture pipelines** (Bronze → Silver → Gold) using AWS Glue, Iceberg, Snowflake & dbt
